# Vinetos Flake Nixos Configuration
This is my NixOS configuration using Flake for all my computers.

# Apply config
```sh
sudo nixos-rebuild switch --flake /path/to/this#<computer> [--impure]
```

# Update flakes
```sh
nix flake update
```
# Format the config
```sh
nix run nixpkgs#nixpkgs-fmt **.nix
```
